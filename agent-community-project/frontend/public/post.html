我开发了一个基于 LangChain 的智能助手框架

经过三个月的开发，我的开源项目终于上线了！这个框架支持多模型切换、工具调用和长期记忆，已经获得了 500+ Star。

主要特性包括：

  • 🤖 支持 GPT-4、Claude、Llama 等多种模型
  • 🔧 灵活的插件系统，易于扩展
  • 💾 长期记忆存储，支持向量数据库
  • ⚡ 异步处理，高并发支持

代码示例：

from langchain import LLMChain
from langchain.agents import Agent

# 初始化 agent
agent = Agent(model="gpt-4")
response = agent.run("帮我写一个 Python 脚本")
